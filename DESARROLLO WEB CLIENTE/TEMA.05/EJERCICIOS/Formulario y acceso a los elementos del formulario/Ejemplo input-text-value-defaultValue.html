ENUNCIADO
Tenemos un formulario formado por el siguiente input tipo texto:
input type="text" id="myText" value="Mickey"

Y el siguiente input tipo reset
input type="reset" value="reset"

Fuera del formulario crea estos 3 botones que hagan lo siguiente:
a) Botón llamado "¿Ha cambiado el valor?": que indique si el usuario ha introducido algún valor en el input.
b) Botón llamado "Cambiar propiedad defaultValue": que modifique dicho valor a Goofy
c) Botón llamado "Imprime valores": que muestre los valores value y defaultValue del input text.

PARA PROBAR:
1- Haz click en el botón Valores, te saldrá el valor de las propiedades defaultValue (value del HTML-Mickey) y value (el valor que ha introducido el usuario-Mickey porque no ha introducido ninguno)
2- Escribe algo en el input y vuelve a pulsar el botón Valores, te saldrá como defaultValue Mickey, y como value lo introducido por el usuario
3- Pulsa el botón Cambiar propiedad default, te saldrá como defaultValue el nuevo valor (Goofy) y como value lo introducido por el usuario
4- Pulsa el botón reset del formulario (que va a poner todos sus elementos al estado por defecto) y pulsa el botón Valores, te saldra el value y defaultValue igual a Goofy
